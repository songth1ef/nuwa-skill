> - 罗斯·特里尔《毛泽东传》（Ross Terrill, *Mao: A Biography*, Stanford University Press, 1999）
> - 菲利普·肖特《毛泽东传》（Philip Short, *Mao: A Life*, Henry Holt, 2000）
> - 金冲及主编《毛泽东传 1893-1949》（中央文献出版社, 1996）

## 第一阶段：成长与启蒙（1893-1918）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1893 | 12月26日生于湖南湘潭韶山冲，父亲毛贻昌为富裕农民 | 农村出身奠定了对农民生活的直觉理解，与后来重视农民革命力量直接相关 |
| 1902-1906 | 入私塾读书，熟读《三字经》《论语》《孟子》等经典 | 儒学根底——特别是"知行合一"的实学传统，成为后来哲学思想的底层结构 |
| 1906-1909 | 辍学在家务农，与父亲发生激烈冲突；私下阅读《水浒传》《三国演义》《西游记》等 | **关键节点**：从小说中获得"造反有理"的朴素正义观；与父亲的对抗经历形成了"斗争"意识的原初体验。据斯诺记录，毛泽东后来将此称为"第一次罢工"的成功 |
| 1910 | 阅读郑观应《盛世危言》，受维新思想影响；入湘乡东山高等小学堂 | 首次接触现代改良思想，开始关注国家命运。《盛世危言》主张学习西方制度，这是毛泽东思想从传统走向现代的第一步 |
| 1911 | 辛亥革命爆发，投笔从戎加入湖南新军，当兵半年 | 革命实践的初体验；在军队中首次阅读报纸，接触孙中山、社会主义等概念 |
| 1912 | 退伍后在长沙多所学校试读（商业、法政、自修），最终决定在湖南省立图书馆自学半年 | **关键节点**：在图书馆大量阅读亚当·斯密《国富论》、达尔文《物种起源》、赫胥黎《天演论》（严复译）、穆勒《论自由》（严复译《群己权界论》）、卢梭《社会契约论》、孟德斯鸠《法意》。自学期建立了广泛的知识框架，形成了"博览-比较-取舍"的读书方法 |
| 1913-1918 | 入湖南省立第一师范学校（五年半学制）| **思想形成的关键时期**，详见下方 |
| 1913-1914 | 师从杨昌济学习伦理学、哲学，精读泡尔生《伦理学原理》并做大量批注 | 杨昌济引导毛泽东接触德国唯心主义哲学和宋明理学。毛泽东在泡尔生书上的批注（约12000字）显示他此时形成了"宇宙大本大源"的世界观——认为精神意志是改造世界的根本力量。这是其后来唯意志论倾向的源头 |
| 1915 | 组织学生社团，广泛通信讨论学问；阅读《新青年》杂志 | 开始实践"以文会友"的组织方法。陈独秀的《新青年》打开了新文化运动的窗口 |
| 1917年4月 | 在《新青年》发表《体育之研究》（署名"二十八画生"）| 首篇公开发表的文章。提出"文明其精神，野蛮其体魄"，体现了身心并重、知行合一的思想。文中"与天奋斗""与人奋斗"的措辞预示了后来的斗争哲学 |
| 1917-1918 | 利用暑假与萧子升徒步游历湖南五县，身无分文靠写对联和乞讨生活 | **关键节点**：这是"调查研究"方法论的原初实践——通过亲身体验了解社会底层。毛泽东后来多次提及这段经历对他了解中国农村的重要性 |
| 1918年4月 | 与蔡和森等人创立新民学会 | 第一个由毛泽东主导建立的组织。新民学会成为他实践组织领导力的平台，也是后来中共湖南组织的人才摇篮 |

### 此阶段阅读图谱
- 中国传统：四书五经、《资治通鉴》、《史记》、宋明理学
- 西方译著：严复译《天演论》《群己权界论》《原富》，泡尔生《伦理学原理》
- 新文化：《新青年》杂志、梁启超文集
- 思想底色：**改良主义 + 唯心主义 + 民族主义**

## 第二阶段：马克思主义转向（1918-1921）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1918年8月-1919年3月 | 赴北京，经杨昌济介绍在北京大学图书馆任助理员（月薪8元）| **关键节点**：在李大钊主管的图书馆工作，接触到马克思主义文献。但此时毛泽东自述"思想是自由主义、民主改良主义、空想社会主义等观念的大杂烩"（斯诺记录）。北大经历让他感受到知识精英的傲慢，强化了反精英主义倾向 |
| 1919年4-7月 | 回到长沙，投入五四运动，创办《湘江评论》| **关键节点**：《湘江评论》创刊号发表《民众的大联合》——这篇文章标志着毛泽东思想的重大转向：从个人意志论转向群众力量论。文中提出"民众联合的力量最强"，已经包含了后来群众路线的萌芽 |
| 1919年7月 | 《湘江评论》被军阀查封（仅出5期）| 第一次切身体会到言论自由的局限性和军阀统治的残暴，推动其从改良转向革命 |
| 1919年10月 | 母亲文七妹病逝 | 个人悲痛与社会苦难交织，强化了改造社会的决心 |
| 1919年12月-1920年4月 | 第二次赴京，大量阅读马克思主义著作 | 集中阅读《共产党宣言》（陈望道译本尚未出，读的是英文摘译和日文转述）、考茨基《阶级斗争》、柯卡普《社会主义史》。毛泽东后来对斯诺说："到1920年夏天，我已经在理论上和在某种程度的行动上，成为一个马克思主义者了" |
| 1920年 | 在长沙创办文化书社，传播新思想；组织驱逐湖南军阀张敬尧运动 | 将理论学习与社会行动结合——这种"边学边干"的模式成为其终身特征 |
| 1920年秋 | 创建长沙共产主义小组；创办湖南自修大学 | 从思想者转变为组织者。自修大学体现了教育与革命相结合的理念 |
| 1920年11月-12月 | 与蔡和森通信讨论"改良还是革命"的根本路线问题 | **关键节点**：明确拒绝无政府主义和改良主义，接受列宁式的暴力革命和无产阶级专政理论。蔡和森来信中详细介绍了俄国革命经验和列宁的建党原则 |
| 1921年1月 | 参加新民学会长沙会员大会，讨论"改造中国与世界"的方法 | 毛泽东在会上系统阐述了五种改造方案并逐一批驳，最终主张"激烈方法的共产主义（列宁的主义）"——这是有记录的最早一次系统性辩论 |
| 1921年7月 | 作为湖南代表出席中国共产党第一次全国代表大会（上海→嘉兴南湖）| 中共创始成员之一，时年27岁。但在一大上并未发挥主导作用 |

### 此阶段阅读图谱
- 马克思主义入门：《共产党宣言》、考茨基《阶级斗争》、柯卡普《社会主义史》
- 无政府主义（后放弃）：克鲁泡特金《互助论》、日本无政府主义刊物
- 李大钊文章：《庶民的胜利》《布尔什维主义的胜利》
- 思想转变：**从自由主义/无政府主义 → 马克思列宁主义**

## 第三阶段：早期革命实践与农民问题的发现（1921-1927）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1921-1923 | 任中共湘区委员会书记，组织安源路矿工人运动和长沙泥木工人罢工 | 早期工人运动实践。但经验表明，城市工人运动在军阀统治下极难持续 |
| 1923年6月 | 出席中共三大，当选中央执行委员会委员 | 进入党的核心层，参与国共合作决策 |
| 1924-1925 | 在广州国民党中央宣传部工作，主编《政治周报》；主办农民运动讲习所 | **关键节点**：主办农民运动讲习所（1926年第六届）是毛泽东将注意力从工人转向农民的标志性事件。他开始系统研究农民问题，这在当时的中共领导层中是异类 |
| 1925年冬 | 回韶山养病期间组织农民协会 | 直接在家乡开展农民运动，从实践中验证"农民是革命主力"的判断 |
| 1926年9月 | 发表《国民革命与农民运动》 | 首次系统论述农民问题对国民革命的决定性意义 |
| 1927年1-2月 | 用32天实地考察湖南湘潭、湘乡、衡山、醴陵、长沙五县农民运动 | **关键节点——调查研究方法论的经典实践**。这次考察是后来"没有调查，没有发言权"方法论的原型 |
| 1927年3月 | 发表《湖南农民运动考察报告》| **里程碑著作**。核心论断：（1）农民是中国革命的主力军；（2）革命不是绣花——"革命是暴动，是一个阶级推翻一个阶级的暴烈的行动"；（3）"矫枉必须过正"的辩证法。这篇报告标志着毛泽东独立于共产国际和中共中央主流路线，形成了以农民为中心的革命理论 |
| 1927年4月 | 蒋介石发动四一二反革命政变 | 国共合作破裂，城市革命路线遭受毁灭性打击，客观上验证了毛泽东农村路线的正确性 |
| 1927年8月7日 | 出席八七会议，提出"枪杆子里面出政权" | **关键节点**：这一论断标志着从"政治斗争为主"转向"军事斗争为主"的路线转变，成为中国革命此后数十年的基本原则 |
| 1927年9月 | 领导秋收起义，起义失败后率部转向井冈山 | **思想史意义极大**：放弃攻打城市（长沙），转向农村建立根据地——这是"农村包围城市"战略的起点，也是与苏联革命模式的根本分歧 |

### 此阶段阅读图谱
- 列宁著作：《国家与革命》《怎么办》《两个策略》
- 农民问题研究：大量地方志、农村调查材料
- 思想关键词：**农民主体论 + 调查研究 + 暴力革命**

## 第四阶段：井冈山与中央苏区——实践-理论循环的黄金期（1927-1934）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1927年10月 | 率秋收起义余部到达井冈山，建立第一个农村革命根据地 | 开创"工农武装割据"模式——在敌人力量薄弱的农村地区建立红色政权 |
| 1928年4月 | 朱德率南昌起义余部与毛泽东在井冈山会师，组建红四军 | 朱毛会师标志着武装斗争进入新阶段。"朱毛"成为红军的代名词 |
| 1928年10月 | 写作《中国的红色政权为什么能够存在？》| **关键理论著作**。从中国社会的特殊矛盾（帝国主义间接统治、军阀割据、经济不平衡）出发，论证农村根据地长期存在的可能性。这是毛泽东运用矛盾分析法的早期范例 |
| 1928年11月 | 写作《井冈山的斗争》 | 详细总结根据地建设经验，提出土地革命、武装斗争、根据地建设"三位一体"的理论框架 |
| 1929年6月 | 在红四军第七次代表大会上落选前委书记 | 遭受党内排挤，被迫离开军队领导岗位。这次挫折促使他深入思考党的建设和军队政治工作 |
| 1929年12月 | 主持古田会议，起草《古田会议决议》 | **关键节点**：确立"党指挥枪"原则和思想建党原则。决议系统批判了军队中的单纯军事观点、极端民主化、流寇思想等错误，建立了人民军队政治工作的基本框架。这是毛泽东组织理论的奠基性文件 |
| 1930年1月 | 写作《星星之火，可以燎原》（致林彪的信）| 批判"红旗到底打得多久"的悲观论调，提出中国革命高潮即将到来的判断。**方法论意义**：展示了从全局看局部、从趋势看现状的战略思维方式 |
| 1930年5月 | 写作《反对本本主义》（原题《调查工作》）| **方法论里程碑**。提出"没有调查，没有发言权"——这是毛泽东认识论的核心命题。文章批判了党内不做调查研究、照搬书本和上级指示的教条主义。同时提出"中国革命斗争的胜利要靠中国同志了解中国情况"——反对盲从共产国际 |
| 1930年10月-1931年9月 | 指挥红军取得第一至第三次反"围剿"胜利 | **实践验证理论**：运用"诱敌深入""集中优势兵力各个击破"等战术原则。这些军事胜利为毛泽东的军事思想奠定了实践基础 |
| 1930-1933 | 在中央苏区进行大量农村调查：寻乌调查、兴国调查、长冈乡调查、才溪乡调查等 | **调查研究方法论的系统化**。《寻乌调查》长达8万字，涵盖当地的地理、交通、商业、土地关系、阶级构成等各方面——堪称社会学田野调查的范本。毛泽东提出"调查就是解决问题""不做正确的调查同样没有发言权" |
| 1931年11月 | 中华苏维埃共和国临时中央政府成立，毛泽东任主席 | 获得政权领导地位，但在军事指挥权上逐渐被博古、李德等人排挤 |
| 1932-1934 | 遭受王明"左"倾路线排挤，失去军事指挥权 | **被边缘化的三年**。但这段时间毛泽东大量阅读马列著作和哲学书籍，为延安时期的理论创作积累了储备。据李锐记载，这一时期他集中阅读了列宁《唯物主义和经验批判主义》、恩格斯《反杜林论》等 |
| 1934年10月 | 红军被迫长征 | 第五次反"围剿"失败，证明了脱离实际的教条主义路线的破产 |

## 第五阶段：长征与遵义会议——从边缘到核心（1934-1936）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1935年1月 | **遵义会议** | **中国革命史最关键的转折点之一**。会议确立了毛泽东在红军和党中央的领导地位（实际上是军事领导权）。方法论意义：遵义会议是"实践检验真理"的经典案例——正是第五次反"围剿"的惨败使全党认识到教条主义路线的错误 |
| 1935年1-4月 | 指挥四渡赤水 | 毛泽东军事指挥艺术的巅峰之作。体现了"灵活机动""避实击虚""运动战"的战略思想 |
| 1935年10月 | 到达陕北，长征结束 | 长征的幸存经历成为全党的共同精神财富，也为毛泽东的权威提供了不可撼动的实践基础 |
| 1935年12月 | 写作《论反对日本帝国主义的策略》 | 提出抗日民族统一战线的完整理论，展示了毛泽东在战略判断上的灵活性——从阶级斗争为主转向民族矛盾为主 |
| 1936年7月-10月 | 在保安接受埃德加·斯诺采访 | 斯诺《红星照耀中国》（1937出版）向世界展示了毛泽东和中共的形象。毛泽东向斯诺系统讲述了自己的人生经历——这是了解其早年思想发展的最重要一手资料 |
| 1936年12月 | 写作《中国革命战争的战略问题》 | **军事辩证法的系统总结**。从战争的一般规律到中国革命战争的特殊规律，从战略防御到战略进攻，系统阐述了矛盾的特殊性原理在军事领域的应用 |

## 第六阶段：延安时期——理论创作的黄金时代（1936-1945）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1937年7-8月 | 在抗日军政大学讲授并写作《实践论》《矛盾论》 | **哲学思想的最高峰**。这两篇著作是毛泽东对马克思主义哲学的最重要贡献 |
| — | **《实践论》**（1937年7月） | 副标题"论认识和实践的关系——知和行的关系"。核心论点：（1）认识来源于实践；（2）认识经历从感性到理性的飞跃；（3）理性认识必须回到实践中检验——"实践、认识、再实践、再认识"的无限循环。**写作背景**：针对党内教条主义（照搬苏联经验）和经验主义（拒绝理论学习）两种错误倾向 |
| — | **《矛盾论》**（1937年8月） | 核心论点：（1）矛盾的普遍性与特殊性；（2）主要矛盾和次要矛盾；（3）矛盾的主要方面和次要方面；（4）对抗性矛盾和非对抗性矛盾的区分。**写作背景**：为统一全党的思维方法，纠正教条主义的"一刀切"思维。毛泽东此前集中阅读了苏联哲学教科书（西洛可夫等《辩证唯物论教程》、米丁《辩证唯物论与历史唯物论》）并做了大量批注 |
| 1937年 | 阅读并批注大量哲学著作 | 据统计，1936-1937年间毛泽东阅读了10余本哲学著作，留下约26000字批注。这是他一生中最集中的理论学习期 |
| 1938年5-6月 | 写作并发表《论持久战》和《抗日游击战争的战略问题》 | **战略预见的杰作**。在全国弥漫速胜论和亡国论两种情绪时，毛泽东提出抗日战争将经历"战略防御—战略相持—战略反攻"三个阶段。**方法论意义**：（1）全面分析中日双方的矛盾（敌强我弱，但敌小我大、敌退步我进步、敌寡助我多助）；（2）抓住主要矛盾的转化规律做长期预判。写作前毛泽东研读了克劳塞维茨《战争论》 |
| 1938年10月 | 在六届六中全会作《论新阶段》报告，提出"马克思主义中国化" | **关键概念的诞生**。毛泽东明确提出："离开中国特点来谈马克思主义，只是抽象的空洞的马克思主义。因此，马克思主义的中国化，使之在其每一表现中带着中国的特性，即是说，按照中国的特点去应用它，成为全党亟待了解并亟须解决的问题。" |
| 1939年 | 写作《〈共产党人〉发刊词》《中国革命和中国共产党》《新民主主义论》 | 系统阐述中国革命的性质、对象、任务、动力和前途。《新民主主义论》提出了新民主主义革命理论的完整框架——这是毛泽东对中国革命理论的最系统贡献 |
| 1941年5月 | 作《改造我们的学习》报告 | 整风运动的理论先声。批判主观主义（教条主义和经验主义），提倡理论联系实际 |
| 1941年9月-10月 | 主持9月政治局扩大会议，系统清算王明路线 | 通过检讨历史路线错误，确立了毛泽东思想在全党的指导地位 |
| 1942年2月 | 作《整顿党的作风》和《反对党八股》两个报告，延安整风运动全面展开 | **关键节点——方法论的制度化**。整风运动将"调查研究""理论联系实际""批评与自我批评"从个人方法论提升为全党的工作方法和组织制度。整风确立了"惩前毖后，治病救人"的党内斗争原则 |
| 1942年5月 | 发表《在延安文艺座谈会上的讲话》 | 提出"文艺为工农兵服务"的方针。方法论上强调"源于生活、高于生活"——文艺创作必须以社会实践为基础 |
| 1943年6月 | 写作《关于领导方法的若干问题》 | **群众路线方法论的经典表述**："从群众中来，到群众中去"——将群众的分散意见集中起来，再到群众中去检验。这实际上是认识论（实践论）在组织工作中的具体应用 |
| 1945年4月 | 在七大作《论联合政府》政治报告；七大确立"毛泽东思想"为党的指导思想 | 毛泽东思想被正式确立。七大总结了三大作风：理论联系实际、密切联系群众、批评与自我批评 |

### 延安时期阅读图谱
- 马列经典：《资本论》（部分）、列宁《哲学笔记》《帝国主义论》
- 哲学：西洛可夫《辩证唯物论教程》、米丁《辩证唯物论与历史唯物论》、艾思奇《大众哲学》、李达《社会学大纲》
- 军事：克劳塞维茨《战争论》、《孙子兵法》
- 中国历史：《资治通鉴》（反复阅读）、《二十四史》（选读）
- 思想特征：**唯物辩证法 + 实践认识论 + 群众路线 + 中国化**

## 第七阶段：解放战争与建国（1945-1949）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1945年8月 | 赴重庆与蒋介石谈判（43天） | 展示了战略灵活性——一面准备战争，一面争取和平的政治主动权 |
| 1946年8月 | 对斯特朗谈话，提出"一切反动派都是纸老虎" | 战略藐视与战术重视相结合的辩证思维。"在战略上藐视敌人，在战术上重视敌人" |
| 1947年12月 | 在十二月会议作《目前形势和我们的任务》报告 | 提出新民主主义革命的三大经济纲领和十大军事原则 |
| 1948年 | 写作一系列军事电报指挥三大战役（辽沈、淮海、平津）| 军事指挥艺术的集大成——在运动中消灭敌人的有生力量。体现了"集中优势兵力""先打分散孤立之敌"等原则 |
| 1949年3月 | 在七届二中全会作报告 | **关键节点**：在即将取得全国胜利时提出"两个务必"——"务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风；务必使同志们继续地保持艰苦奋斗的作风。"这是对胜利后可能出现的骄傲和腐化的预警 |
| 1949年6月 | 发表《论人民民主专政》 | 阐述新中国的国体理论——人民民主专政。系统总结了中国革命的基本经验 |
| 1949年10月1日 | 中华人民共和国成立 | 从革命者变为执政者——角色的根本转变 |

## 第八阶段：建国初期的探索与务实（1949-1956）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1949-1952 | 领导土地改革、镇反、三反五反、抗美援朝 | 新政权的巩固期。这些运动展示了毛泽东运用群众运动进行社会改造的方法——既有效但也埋下了以运动代替制度建设的隐患 |
| 1953年 | 提出过渡时期总路线：逐步实现社会主义工业化和三大改造 | 从新民主主义转向社会主义的战略决策。初期设想是15年或更长的渐进过渡 |
| 1955年7月 | 作《关于农业合作化问题》报告，批判"小脚女人"式的右倾保守 | **思想转向的信号**：从渐进路线转向加速推进。毛泽东开始认为群众的社会主义积极性被干部的保守主义压制了 |
| 1956年2月 | 赫鲁晓夫在苏共二十大作秘密报告批判斯大林 | **关键外部冲击**：促使毛泽东深入思考社会主义国家的内部矛盾问题，也增强了他对苏联模式的怀疑 |
| 1956年4月 | 作《论十大关系》报告 | **重要理论创新**。运用矛盾分析法系统论述社会主义建设中的十对矛盾（重工业与轻工业、沿海与内地、中央与地方等），提出"以苏为鉴"走中国自己的建设道路。这是毛泽东最务实、最具建设性的文本之一 |
| 1956年9月 | 中共八大召开，确认主要矛盾转变为"先进的社会制度与落后的社会生产力之间的矛盾" | 八大路线是集体智慧的结晶。毛泽东此时基本认同这一判断，但不久后即动摇 |

## 第九阶段：从百花运动到大跃进——激进化的开始（1957-1960）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1957年2月 | 作《关于正确处理人民内部矛盾的问题》讲话 | **重要理论贡献**。首次系统区分"敌我矛盾"和"人民内部矛盾"两类不同性质的矛盾，提出用"团结—批评—团结"的公式处理人民内部矛盾。这是对马克思主义国家理论的重要发展。但后来这一理论在反右运动中被严重扭曲 |
| 1957年春 | 鼓励"百花齐放、百家争鸣" | 鼓励知识分子提意见，试图用民主方式解决人民内部矛盾 |
| 1957年6月起 | 反右运动扩大化，55万人被划为右派 | **关键转折点**：从鼓励批评到大规模整肃。知识分子批评的激烈程度超出毛泽东预期，促使他将"人民内部矛盾"的边界大幅收窄。此后，"阶级斗争"重新成为主导叙事 |
| 1958年 | 发动大跃进运动，提出"超英赶美"、人民公社化 | **激进化的高峰**。毛泽东认为可以通过群众运动的高涨热情突破经济发展的客观规律。这是其唯意志论倾向的极端表现——用政治激情替代科学规划 |
| 1958年11月-1959年 | 第一次郑州会议起，开始纠左 | 面对大跃进的严重问题，毛泽东开始部分纠正。他在读苏联《政治经济学教科书》时做了大量批注，承认"价值规律是一个伟大的学校" |
| 1959年7月 | **庐山会议**（先纠左，后反右） | **关键转折点**。彭德怀上书批评大跃进问题，毛泽东将其定性为"右倾机会主义"进行批判。庐山会议标志着纠左努力的夭折——此后再无人敢于向毛泽东讲真话。这一事件深刻影响了此后十余年的政治生态 |
| 1959-1961 | 三年困难时期（大饥荒）| 大跃进的灾难性后果。据各方估计，非正常死亡人数在1500万至4500万之间（学术界仍有争议）|
| 1961年 | 发起大兴调查研究之风，亲自和组织多个调查组下乡 | 面对大跃进的失败，重新回到调查研究的方法论。但这次"纠错"是有限度的——承认方法有误，但不触及路线根本 |
| 1962年1月 | 七千人大会，刘少奇作报告总结教训，毛泽东作自我批评 | 部分承认大跃进的错误（"三分天灾七分人祸"的说法即出于此时）。但毛泽东的自我批评是有限的，且他感到自己的权威受到挑战 |

### 此阶段阅读图谱
- 苏联政治经济学教科书（做了大量批注，约10万字）
- 斯大林《苏联社会主义经济问题》
- 反复阅读《资治通鉴》（据身边工作人员回忆，此书读了17遍）
- 思想特征：**唯意志论强化 + 对官僚体制的不信任 + 阶级斗争扩大化**

## 第十阶段：从调整到文化大革命（1962-1969）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1962年9月 | 在八届十中全会上重提阶级斗争："千万不要忘记阶级斗争" | **关键转折**：从经济调整重新转向政治运动。毛泽东判断党内已经出现"修正主义"危险，社会主义社会仍然存在阶级和阶级斗争 |
| 1963-1965 | 发动四清运动（社会主义教育运动）| 试图通过群众运动清除基层干部的腐败和"资本主义倾向"。但运动效果不佳，使毛泽东认为问题的根源在上层 |
| 1963-1964 | 中苏论战，主持撰写"九评"系列文章 | 与苏联的公开决裂。毛泽东认为苏联已经"变修"（和平演变为资本主义），并将这一判断投射到国内——担心中国也会出现赫鲁晓夫式的人物 |
| 1964年 | 提出"培养无产阶级革命事业接班人"的五项条件 | 接班人问题成为核心焦虑。毛泽东日益担忧自己身后中国会改变颜色 |
| 1965年 | 重上井冈山；批判《海瑞罢官》 | 重上井冈山带有强烈的象征意义——回到革命的起点。批判《海瑞罢官》（姚文元文章发表于1965年11月）成为文革的导火索 |
| 1966年5月 | 发动无产阶级文化大革命 | **最激进的思想实验**。核心逻辑：（1）社会主义社会存在资产阶级和资本主义复辟的危险；（2）危险的根源在党内（"走资本主义道路的当权派"）；（3）必须发动群众"自下而上"地揭露和批判党内的修正主义。这是对自己建立的党和国家体制的自我革命，在人类政治史上独一无二 |
| 1966年8月 | 写《炮打司令部——我的一张大字报》；接见百万红卫兵 | 公开打倒刘少奇为代表的"资产阶级司令部"。红卫兵运动的大规模展开 |
| 1966-1968 | 文革高潮：打倒"走资派"、武斗、夺权 | 社会陷入严重动荡。毛泽东试图用"天下大乱达到天下大治"的辩证逻辑为混乱辩护，但实际后果远超其控制能力 |
| 1968年 | 下达"知识青年到农村去"指示 | 上山下乡运动——既是疏散城市红卫兵的务实考虑，也体现了"与工农相结合"的理想主义信念 |
| 1969年4月 | 九大召开，林彪被确定为接班人 | 文革前期的"成果"被制度化。但新的权力格局极不稳定 |

## 第十一阶段：晚年（1969-1976）

| 年份 | 事件 | 思想意义 |
|------|------|----------|
| 1971年9月 | 林彪事件（九一三事件）| **严重打击**。亲手选定的接班人叛逃身亡，这对毛泽东的精神和身体都造成了沉重打击。也从根本上动摇了文革的理论逻辑——如果林彪是坏人，那文革的"成果"还剩什么？ |
| 1972年2月 | 尼克松访华 | **战略外交的杰作**。在中苏关系恶化的背景下，与最大的资本主义国家和解——展示了毛泽东晚年在外交上的战略灵活性与意识形态刚性的鲜明反差 |
| 1973-1974 | 批林批孔运动 | 将批判林彪与批判孔子联系起来。晚年毛泽东越来越多地借用历史典故来表达政治意图——法家与儒家的对立成为其晚年思维的重要框架 |
| 1974年 | 提出"三个世界"理论 | 将世界划分为超级大国（第一世界）、发达国家（第二世界）、发展中国家（第三世界），中国属于第三世界。这是其国际战略思维的最后重要贡献 |
| 1975年 | 邓小平主持全面整顿，后再次被打倒 | 毛泽东在"要稳定"和"要继续革命"之间反复摇摆。先支持邓小平整顿，后又认为邓要"翻文革的案" |
| 1976年1月 | 周恩来逝世 | 失去了最重要的行政合作者 |
| 1976年4月 | 天安门事件（四五运动），民众自发悼念周恩来 | 群众以非官方方式表达对文革极左路线的不满。毛泽东批准定性为"反革命事件" |
| 1976年9月9日 | 毛泽东逝世，享年82岁 | 一个时代的终结 |
| 1976年10月 | "四人帮"被逮捕，文革结束 | 毛泽东身后不到一个月，其政治路线即遭根本性逆转 |
| 1981年 | 中共十一届六中全会通过《关于建国以来党的若干历史问题的决议》 | 官方评价："毛泽东同志是伟大的马克思主义者，是伟大的无产阶级革命家、战略家和理论家。他虽然在'文化大革命'中犯了严重错误，但是就他的一生来看，他对中国革命的功绩远远大于他的过失。他的功绩是第一位的，错误是第二位的。" |

## 思想发展的六次关键转折

| 序号 | 时间 | 转折内容 | 驱动因素 |
|------|------|----------|----------|
| 1 | 1919-1920 | 从自由主义/无政府主义 → 马克思列宁主义 | 五四运动的实践 + 十月革命的示范 + 李大钊等人的影响 |
| 2 | 1927 | 从城市工人运动 → 农村武装割据 | 国共合作破裂 + 秋收起义失败的实践教训 |
| 3 | 1935 | 从被边缘化 → 确立领导地位 | 第五次反"围剿"失败证伪了教条路线 |
| 4 | 1937-1942 | 从实践经验 → 系统化理论（实践论、矛盾论、整风） | 延安的相对稳定环境 + 总结十年内战经验的需要 |
| 5 | 1957-1958 | 从务实建设 → 激进运动（反右、大跃进） | 对官僚化的焦虑 + 知识分子批评的刺激 + 赶超心态 |
| 6 | 1966 | 发动文化大革命——对自己建立的体制进行自我革命 | 对"修正主义"复辟的恐惧 + 林彪事件后的孤独与焦虑 |

| 转变（1918-1921） | 《共产党宣言》、考茨基、柯卡普、李大钊文章 | 转向马克思主义 |

| 延安（1936-1945） | 苏联哲学教科书、克劳塞维茨、《孙子兵法》、李达《社会学大纲》 | 哲学体系化；军事理论化 |
| 建国后（1949-1976） | 《资治通鉴》（反复）、《二十四史》、苏联政治经济学教科书、鲁迅全集、古诗词 | 帝王术与革命理论的交织；晚年日益回归中国传统 |

> 毛泽东一生手不释卷。据中南海图书管理员统计，其个人藏书近10万册，床头常年堆满书籍。他尤其偏爱《资治通鉴》和《二十四史》，晚年视力衰退后仍坚持让人读给他听。
